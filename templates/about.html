About This Project

PassCheck is a Flask-based web tool to check the strength of your passwords. It uses scoring logic to detect weak, medium, or strong passwords and offers improvement tips.

Features

  • 💪 Password strength checker with live suggestions
  • 🔐 Strong password generator
  • 📊 Last 5 password history (session-based)
  • 🌍 Works offline, fast and secure

Tech Stack

Python 🐍 · Flask ⚙️ · HTML/CSS 🎨 · JavaScript
